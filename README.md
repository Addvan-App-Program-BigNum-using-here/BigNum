# 1. 프로젝트 소개

### BigNum을 이용하여 암호 알고리즘을 만들고 Application 단에서 활용해보자!
공개키 암호에서 주로 사용되는 RSA, ECC와 같은 암호 알고리즘에서 필요한 큰 수를 다루는 라이브러리인 BigNum을 이용하여 암호 알고리즘을 만들어보고, 이를 Application 단에서 활용해보는 프로젝트입니다.
